# Zaawansowane Programowanie Obiektowe 78404

Prosty program konsolowy w C#. Służy do obsługi wypożyczalni rowerów: wyświetlania rowerów, wypożyczania, zwrotów, zmiany ceny i podglądu danych.

Aplikacja konsolowa była kompilowana i testowana na .NET 9.0.

Funkcje

- pokazuje wszystkie rowery
- pokazuje dostępne rowery
- wypożycza rower
- przyjmuje zwrot roweru
- zmienia cenę dzienną roweru
- wyświetla właściwości obiektu przez refleksję
- uruchamia prostą operację asynchroniczną

Uruchomienie

Wymagania:
- zainstalowany .NET SDK 9.0
- projekt typu Console App ustawiony na net9.0

Uruchomienie przez terminal:
1. Wejdź do katalogu projektu.
2. Sprawdź wersję SDK poleceniem dotnet --info.
3. Zbuduj projekt poleceniem dotnet build.
4. Uruchom program poleceniem dotnet run albo dotnet run --project ConsoleApp2.csproj.

Uruchomienie w Visual Studio:
1. Otwórz projekt.
2. Sprawdź, czy framework projektu to .NET 9.0.
3. Skompiluj rozwiązanie.
4. Uruchom program.

Użycie

Program działa z poziomu menu. Użytkownik wpisuje numer opcji i zatwierdza Enterem.
